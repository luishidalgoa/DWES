### Sintaxis Básica en PHP

PHP es un lenguaje de programación ampliamente usado en desarrollo web. El código PHP se inserta entre etiquetas `<?php` y `?>`, y cada instrucción termina con un punto y coma (`;`). Las variables en PHP se declaran con el prefijo `$` seguido del nombre de la variable, como `$nombre`. También permite el uso de estructuras de control como `if`, `while`, y `for`. Las funciones se declaran con `function nombreFuncion()`, y los comentarios se añaden con `//` para una línea o `/* ... */` para múltiples líneas. Aquí un ejemplo básico:

```php
<?php
$saludo = "¡Hola, PHP!";
echo $saludo; // Imprime el saludo
?>
```
